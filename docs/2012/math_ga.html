Let \(\{a_n\}\) be a sequence with an initial term of \(1\), and let \(S_n = \displaystyle\sum_{k\;\!=\;\!1}^{n}a_k\), such that
\(n S_{n+1} = (n+2)S_n + (n+1)^3\;\;\) (\(n \geq 1\)).
The following is a part of a process computing the general term \(a_n\).
\(S_{n+1} = S_n + a_{n+1}\) for positive integers \(n\), so
\(na_{n+1} = 2 S_n + (n+1)^3 \).
ⓐ
For integers \(n \geq 2\), we have
\((n-1)a_n = 2 S_{n-1} + n^3 \).
ⓑ
Subtracting ⓑ from ⓐ, we get
\(n a_{n+1} = (n+1) a_n + \fbox{\(\;(\alpha)\;\)} \).
Dividing both sides with \(n(n+1)\), we get
\(\dfrac{a_{n+1}}{n+1} = \dfrac{a_n}{n} + \dfrac{\fbox{\(\;(\alpha)\;\)}}{n(n+1)} \).
Let \(b_n = \dfrac{a_n}{n}\). Since
\(b_{n+1} = b_n + 3 + \fbox{\(\;(\beta)\;\)}\;\;\) (\(n \geq 2\)),
we have
\(b_n = b_2 + \fbox{\(\;(\gamma)\;\)}\;\;\) (\(n \geq 3\)).
\(\vdots \)
Let \(f(n)\), \(g(n)\) and \(h(n)\) be the correct expressions for \((\alpha)\), \((\beta)\) and \((\gamma)\). What is the value of \(\dfrac{f(3)}{g(3)h(6)}\)? [4 points]
  1. \(30\)
  2. \(36\)
  3. \(42\)
  4. \(48\)
  5. \(54\)
